# Shop-Shop Redux-React

## Description
Storefront that allows you to put multiple things in cart and "purchase" using third party Stripe to take payments. Store's order history API using state and store of redux. 

## Built With 
* CSS
* Javascript
* HTML
* Express
* Mongoose
* Graph SQL
* Stripe
* React
* Redux
